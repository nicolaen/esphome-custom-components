# ESPHome custom components
This repository provides custom components that you can use in ESPHome

## How to use
All components must be placed in your esphome configuration directory under `custom_components`.
```
esphome
├── custom_components
│   ├── component_1
│   │   ├── __init.py__
│   │   ├── component_1.cpp
│   │   ├── component_1.h
│   │  ...
│   ├── component_2
│   │   ├── __init.py__
│   │   ├── component_2.cpp
│   │   ├── component_2.h
│   │  ...
│  ...
```
